# sdkworkfs 架构文档

本目录用于定义 `sdkworkfs` 的产品、架构、API、缓存、Git server、节点守护进程和部署标准。

## 当前基线

自本轮重构起，文档基线统一为：

- 全 Rust
- 多二进制
- 多 crate
- 管理 API / 运行时 API / 节点 API / Git 协议分层
- 完整 Git server 作为一等能力

本目录只维护当前 Rust 基线文档。
